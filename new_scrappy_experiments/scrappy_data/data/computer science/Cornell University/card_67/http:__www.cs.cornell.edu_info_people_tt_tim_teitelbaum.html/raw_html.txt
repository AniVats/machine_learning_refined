Making the cyber-world a safer place includes uncovering bugs and vulnerabilites in software that can be exploited by attackers around the globe.</p>\n<p>To protect software from attacks optimally, we must analyze both source code and machine code during software development. Analzying source code is critical for developers to understand the vulnerabilities that may exist in the code they are writing. Analyzing machine code allows organizations to scan components of code, such as libraries or full applications, using stripped executables. By analyzing the code as it will be executed in the final application, we are able to achieve deeper insight than would be attainable from analysis of source code alone.</p>\n\n<p>Dr. Teitelbaum retired from Cornell in June 2011 to devote his time to <a href=\"http://www.grammatech.com\">GrammaTech, Inc.</a>, where he is the CEO. He continues to pursue both his research interests and the transition of research to commercial products.</p>\n\n\n<h2>Honors and Awards:</h2>\n<p><strong>2010</strong><br>ACM SIGSOFT Retrospective Impact Paper Award for the 1984 paper, \"The Synthesizer Generator\" co-authored with Dr. Thomas Reps</p>\n<p><strong>2003</strong><br>Recognized as a \u201cHighly Cited Researcher\u201d in the field of Computer Science by the Institute for Scientific Information.</p>\n<p><strong>1996</strong><br>Dean's Award for Outstanding Teaching</p>\n<p><strong>1996</strong><br>College of Engineering Teaching Award</p>\n<p><strong>1986</strong><br>Dean's Prize for Innovation in Undergraduate Teaching</p>\n\n\n</div></div></div></div></div>\n<hr>\n\n<div id=\"footer\">\n<div id=\"footer-content\">\u00a92013 <a href=\"http://www.cornell.edu/\">Cornell \nUniversity</a> </div></div>\n</body></html>"
