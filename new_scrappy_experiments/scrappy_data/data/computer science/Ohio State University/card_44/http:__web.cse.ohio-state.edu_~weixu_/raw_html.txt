margin-right: 1em\">\n\t\t\t\t<div class=\"boxed\">\n\t\t\t\t\t<p><a href=\"files/pic_semantics.png\"><img border=\"1px\" hspace=\"10\" src=\"files/pic_semantics.png\" style=\"float: right;\" vspace=\"5\" width=\"320px\"></a>\n\t\t\t\t\t</p>\n\n\n\t\t\t\t\t<h5><strong>Natural Language Understanding / Semantics</strong>\n\t\t\t\t\t</h5>\n\n\n\t\t\t\t\t<p>My approach to natural language understanding is learning and modeling paraphrases on a much larger scale and with a much broader range than previous work, essentially by developing more robust machine learning models and leveraging social media data. These paraphrase can enable natural language systems to handle errors (e.g., \u201ceverytime\u201d \u2194 \u201cevery time\u201d), lexical variations (e.g., \u201coscar nom\u2019d doc\u201d \u2194 \u201cOscar-nominated documentary\u201d), rare words (e.g \u201cNetsBulls series\u201d \u2194 \u201cNets and Bulls games\u201d), and language shifts (e.g. \u201cis bananas\u201d \u2194 \u201cis great\u201d). We designed a series of unsupervised and supervised learning approaches for paraphrase identification in social media data (also applicable to question/answer pairs [COLING'18] for QA systems), ranging from neural network models <a href=\"publications/COLING_2018_sentence_pair_modeling.pdf\">[COLING'18]</a> <a href=\"publications/EMNLP2017_growing_dataset.pdf\">[NAACL'18a]</a> to multi-instance learning <a href=\"publications/tacl2014-extracting-paraphrases-from-twitter.pdf\">[TACL'14]</a> <a href=\"https://aclweb.org/anthology/D16-1030.pdf\">[EMNLP'16]</a>, and crowdsourcing large-scale datasets <a href=\"publications/semeval_pit_2015_overview.pdf\">[SemEval'15]</a> <a href=\"publications/EMNLP2017_growing_dataset.pdf\">[EMNLP'17]</a>. \n\n\t\t\t\t</p></div>\n\t\t\t\t<!-- end boxed-->\n\n\t\t\t\t<div class=\"boxed\">\n\t\t\t\t\t<p><a href=\"files/pic_readability.png\"><img border=\"1px\" hspace=\"10\" src=\"files/pic_readability.png\" style=\"float: left;\" vspace=\"5\" width=\"260px\"></a>\n\t\t\t\t\t</p>\n\n\n\t\t\t\t\t<h5 style=\"text-align: right\"><strong>Natural Language Generation / Stylistics</strong>\n\t\t\t\t\t</h5>\n\n\n\t\t\t\t\t<p>Many text-to-text generation problems can be thought of as sentential paraphrasing or monolingual machine translation. It faces an exponential search space larger than bilingual translation, but a much smaller optimal solution space due to specific task requirements. I advocate for a text-to-text generation framework, building on top of machine translation technologies. My recent work uncovered multiple serious problems in previous research (2010 and 2014) on <a href=\"publications/tacl2015-text-simplification-opinion.pdf\">text simplification [TACL'15]</a> , designed a new tunable metric SARI <a href=\"publications/tacl2016-smt-simplification.pdf\">[TACL'16]</a> which is effective for evaluation and as a learning objective for training (now implemented by the Google AI group <a href=\"https://github.com/tensorflow/tensor2tensor/blob/master/tensor2tensor/utils/sari_hook.py\">in TensorFlow</a>), optimized syntax-based machine translation models <a href=\"publications/tacl2016-smt-simplification.pdf\">[TACL'16]</a>, created pairwise neural ranking models to achieve new state-of-the-art performance on lexical simplification <a href=\"publications/EMNLP2018_neural_ranking_and_word_complexity_lexicon_preprint.pdf\">[EMNLP'18]</a>, and studied document-level simplification <a href=\"\">[AAAI'20]</a>. I am interested in text generation for <a href=\"http://aclweb.org/anthology/C/C12/C12-1177.pdf\">style transfer [COLING'12]</a> and stylistics in general (e.g. historic \u2194 modern, non-standard \u2194 standard <a href=\"http://aclweb.org/anthology/W/W13/W13-2515.pdf\">[BUCC'13]</a>, feminine \u2194 masculine <a href=\"publications/AAAI_2016_gender_paraphrase.pdf\">[AAAI'16]</a>).\n\t\t\t\t\t</p>\n\t\t\t\t</div>\n\t\t\t\t<!-- end boxed-->\n\n\t\t\t\t<!-- end boxed-->\n\t\t\t</div>\n\t\t</div>\n\n\n\t\t<div class=\"content\">\n\t\t\t<h5 class=\"text-primary\" id=\"publications\">Publications</h5>\n\n\n\t\t\t<ul>\n\t\t\t\t\n\t\t\t\t<li>\n\t\t\t\t\t<b><a href=\"publications/AAAI-ZhongY.9975.pdf\" id=\"smt-simple\" name=\"smt-simple\" style=\"font-style: italic;
