Research</h4></dt>\n    <dd><ul>\n\t<li><p><span class=\"reslabel\">MJ</span>: \n\tI developed MJ, a system which applies meta-level compilation techniques\n\tto Java.  We have used meta-level compilation as a static analysis\n\ttool for finding bugs in large software systems.  It found thousands of bugs in \n\tthe Linux kernel, for instance.</p>\n\t<p>Of course, in a system written in a weakly-typed language such as C, bugs are a\n\tdime a dozen.  The challenge in MJ is to apply meta-level compilation\n\tto a type-safe language from which simple bugs, such as dangling pointers, have already \n\tbeen eliminated.\n\t</p>\n        </li>\n\t<li><p><span class=\"reslabel\">KaffeOS</span>: \n\t    Single-language runtime systems, in the form of Java virtual machines, are \n\t    widely deployed platforms for executing untrusted mobile code. These runtimes \n\t    provide some of the features that operating systems provide: inter-application memory \n\t    protection and basic system services. They do not, however, provide the ability to \n\t    isolate applications from each other, or limit their resource consumption. KaffeOS is \n\t    a Java runtime system that provides these features. The KaffeOS architecture takes \n\t    many lessons from operating system design, such as the use of a user/kernel boundary, \n\t    and employs garbage collection techniques, such as write barriers.\n\t    While KaffeOS has not been released, a successor project, JanosVM is available\n\t    <a href=\"http://www.cs.utah.edu/flux/janos/\">here.</a></p>\n\t</li>\n\t<li><p><span class=\"reslabel\">DataScript</span>:\n\t    DataScript is a language to describe and manipulate binary data formats as types. \n\t    DataScript consists of two components: a constraint-based specification language that \n\t    uses DataScript types to describe the physical layout of data and a language binding \n\t    that provides a simple programming interface to script binary data. A DataScript compiler \n\t    generates Java libraries that are linked with DataScript scripts.\n\t    A release of the software is available \n\t    at <a href=\"http://datascript.sourceforge.net/\">SourceForge</a>.\n\t    </p>\n\t</li>\n\t<li><p><span class=\"reslabel\">OSKit</span>:\n\t    The OSKit is a framework and a set of 34 component libraries oriented to operating \n\t    systems, together with extensive documentation. By providing in a modular way not only \n\t    most of the infrastructure \"grunge\" needed by an OS, but also many higher-level \n\t    components, the project's goal is to lower the barrier to entry to OS R&amp;D and to lower \n\t    its costs. The OSKit makes it vastly easier to create a new OS, port an existing OS to \n\t    the x86, or enhance an OS to support a wider range of devices, file system formats, \n\t    executable formats, or network services. \n\n\t    For language researchers and enthusiasts, the OSKit lets them concentrate on the real \n\t    issues raised by using advanced languages inside operating systems, such as Java, Lisp, \n\t    Scheme, or ML--- instead of spending six months or years groveling inside ugly code \n\t    and hardware.\n\t\tThe <a href=\"http://www.cs.utah.edu/flux/oskit/\">OSKit homepage.</a></p>\n\t</li>\n\t<li><p><span class=\"reslabel\">Kaffe</span>:  I was a contributor to \n\t\tthe <a href=\"http://www.kaffe.org\">Kaffe project</a>, an open source Java virtual
